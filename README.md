# pushOnline
上线脚本

# 使用方法

上传 github 时我已修改文件执行权限，如果权限不够请给予 proOnline.sh 脚本 u+x 或者 a+x 权限

1. clone项目到本地
2. 软件一个你喜欢的名字到 proOnline.sh 例如：ln -s /Users/Carl/Tools/pushOnline/proOnline.sh /usr/local/bin/in
3. 在开发分支使用 in 命令，上线
